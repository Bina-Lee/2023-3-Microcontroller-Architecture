PC VS ES
공통점
-주 구성요소 CPU Memory IO
-하드웨어 제어를 위한 소프트웨어 필요
-표준 통신 방식들 채택
	RS-232C LAN WLAN USB...etc
차이점
-General Purpose vs Application-Specific
-IO방식 : Keyboard Mouse Display vs LED Segment Botton
-개발환경 : 해당PC에서 가능 vs Host PC for Compile Loading and Monitoring
	Compile : translate : Language -> Binary

Why ES??
PC와 비슷하지만 사용목적이 다름
-간단한 구성과 동작원리로 컴퓨터 전반적인 동작 이해 및 개발 가능
Low Price로 간단한 Application 구성, 제어가능
각종 사물을 컴퓨팅 가능한 통신 장치로 만들수있음
-IoT...etc
